Проект DHCP и DNS серверов

Введение

Этот проект разработан для симуляции DHCP и DNS серверов. Он включает в себя реализацию основных функций серверов, таких как предложение IP-адресов и разрешение доменных имен.

Автор

Разработчик: aleks15868
Почта: ml963.nosp@m.852@.nosp@m.yande.nosp@m.x.ru

Конфигурация

Конфигурация серверов загружается из файла configuration.json. В этом файле содержатся следующие параметры:

  • IP-адрес DHCP: 192.168.2.3
  • Маска подсети DHCP: 255.255.255.0
  • IP-адрес DNS: 192.168.2.4
  • Время жизни IP-адреса: 7200 секунд
  • Начальный диапазон IP: 192.168.2.5 - 192.168.2.100

Основные файлы проекта

  • dhcp_server.py: Реализует функционал DHCP сервера.
  • dns_server.py: Реализует функционал DNS сервера.
  • configuration.json: Содержит параметры конфигурации для DHCP и DNS серверов.
  • domain_dns_name_ip.json: Содержит список доменов и их IP для DNS сервера.
